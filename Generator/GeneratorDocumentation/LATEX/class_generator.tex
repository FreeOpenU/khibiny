\subsection{Detailed Description}
A \hyperlink{class_generator}{Generator} class. 

\hyperlink{class_generator}{Generator} class creates a generator object whose functions output strings. These output strings are strung together to be output into a file. \hyperlink{class_generator}{Generator} class is initialized with a string for a filename to use for output. The default filename is \char`\"{}\+Information.\+txt\char`\"{}
